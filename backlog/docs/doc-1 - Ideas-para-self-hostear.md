## Ideas para self-hostear

Lista de aplicaciones consideradas para añadir al homelab, con valoración inicial.

### Casi seguro

| App | Notas |
|-----|-------|
| **Obsidian** | Comparado con AppFlowy, AFFiNE, Anytype... pero el hecho de que sea markdown plano y se pueda usar con Claude Code le da superpoderes. |
| **Wger** | Seguimiento de ejercicio/fitness. Mola mucho. |
| **Your spotify** | Para ver stats de spotify https://github.com/Yooooomi/your_spotify?tab=readme-ov-file

### Valorar

| App | Notas |
|-----|-------|
| **Dawarich** | Tracking de ubicación. Se integra con Immich. |
| **Mealie** | Gestión de recetas de comida. |
| **OpenClaw** | Asistente personal de IA self-hosted que puede ejecutar tareas como gestionar calendarios, navegar por la web y ejecutar comandos de terminal. |
| **Firefly III** | Finanzas personales. Considerar alternativas como plaintext accounting. |
| **n8n** | Automatización de workflows. |
| **Siftly** | Gestor de bookmarks de Twitter/X con categorización por IA, búsqueda semántica y visualización en mindmap interactivo. Stack: Next.js + SQLite + Prisma + Anthropic SDK. Todo local excepto llamadas a la API de IA. [GitHub](https://github.com/viperrcrypto/Siftly) |
| **norish** | Recetas: https://github.com/norish-recipes/norish

### Probablemente no
